# Technology Stack

## Core Framework & Language
- **Next.js 14 (App Router):** The primary React framework for server-side rendering, routing, and API handling.
- **TypeScript:** Ensuring type safety and better developer experience across the codebase.

## Frontend & Styling
- **React 18:** The core library for building user interfaces.
- **Tailwind CSS:** For rapid, utility-first styling and responsive design.
- **Lucide React:** A comprehensive icon library for consistent visual language.
- **Recharts:** For data visualization in the dashboard (e.g., financial summaries).

## Mobile App (React Native)
- **React Native (via Expo):** Cross-platform mobile framework for iOS and Android.
- **React Navigation:** For robust tab and stack-based navigation.
- **Expo Secure Store:** For secure local storage of authentication tokens.
- **Lucide React Native:** Mobile-optimized version of the Lucide icon library.

## Backend & Infrastructure
- **Neon (Serverless Postgres):** The primary relational database for structured data storage.
- **NextAuth.js v5 (Beta):** Handling authentication and authorization with a flexible, secure provider.
- **Vercel Blob:** Used for centralized file storage (e.g., documents, receipts).
- **Grammy:** Telegram Bot framework for notifications and integration.

## Utilities & Security
- **bcryptjs:** For secure password hashing and verification.
- **date-fns:** For consistent and easy date manipulation.
- **clsx & tailwind-merge:** Utilities for dynamic and conflict-free CSS class management.
